## 🚀 Project Overview  

**WanderStay** is a full-stack accommodation booking platform that enables users to **explore, search, and review properties**, while hosts can **manage listings through complete CRUD operations**.  

The platform is built using a scalable **MVC architecture** and follows RESTful design principles to ensure clean and maintainable code.

---

## ✨ Key Features  

### 🔍 Search & Filter System  
- Search properties based on **location**  
- Filter listings by **category**  
- Enhances user experience with quick discovery  

### 🏡 Property Management (CRUD)  
- Hosts can **create, edit, and delete listings**  
- Upload images and detailed property descriptions  
- Structured listing management  

### 👤 Authentication & Authorization  
- Secure user login and signup  
- Password hashing for security  
- Session management using **Express-session** and **MongoDB session store**  
- Role-based access control  

### 💬 Reviews & Ratings  
- Users can post reviews on listings  
- Helps build trust and transparency  

### 🗺️ Interactive Maps  
- Integrated maps to display property locations  
- Improves usability and navigation  

### 📱 Responsive Frontend  
- Fully responsive design  
- Seamless experience across devices  

### ⚡ Backend Architecture  
- Built using **MVC pattern**  
- RESTful APIs for clean communication  
- Organized routing using **Express Router**
